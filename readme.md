> Caputre tweet


## Install

```
$ npm install capture-tweet
```


## Usage

```js
const fs = require('fs');
const captureTweet = require('capture-tweet');

const tweet = 'https://twitter.com/akameco/status/903892994998153216'

captureTweet(tweet).then(img => {
  fs.writeFileSync('tweet.png', img)
})
```
